 Stock Price Analyzer
 
This project analyzes stock prices using Python.
Features:
- Downloads stock data from Yahoo Finance
- Calculates moving average
- Detects stock trend
- Shows statistics
- Displays graph
Requirements:
pip install pandas matplotlib yfinance
Run:
python stock_price_analyzer.py
Example ticker symbols:
-AAPL
-TSLA
-MSFT
-GOOGL
